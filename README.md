# 🤖 ربات دانلودر تلگرام به روبیکا

یک یوزربات پیشرفته که فایل‌ها را از تلگرام دریافت و به صورت خودکار به حساب روبیکا ارسال می‌کند. قابلیت دانلود از لینک مستقیم و پلتفرم‌های ویدیویی مانند یوتیوب، اینستاگرام و تیک‌تاک را نیز دارد.

---

## ✨ امکانات

- 📥 **دریافت فایل از تلگرام** — فایل‌ها را از ربات بگیر و مستقیم به روبیکات برسه
- 🔗 **دانلود از لینک** — یوتیوب، اینستاگرام، توییتر/X، تیک‌تاک، فیسبوک و لینک مستقیم
- 🎞️ **انتخاب کیفیت ویدیو** — برای سایت‌های ویدیویی کیفیت دانلود رو انتخاب کن
- 📦 **ارسال فشرده (زیپ)** — فایل‌ها رو معمولی یا زیپ یا زیپ رمزدار بفرست
- ✂️ **تقسیم فایل‌های بزرگ** — فایل‌های بالای ۱۰۰MB خودکار به پارت‌های ۹۵MB تبدیل می‌شن
- 📊 **نوار پیشرفت زنده** — نمایش درصد، سرعت و حجم در حین دانلود و ارسال
- 🗓️ **تاریخ شمسی** — کپشن فایل‌ها با تاریخ و ساعت فارسی
- 🔒 **دسترسی محدود** — فقط یه کاربر مشخص (خودت) می‌تونه استفاده کنه
- 📈 **آمار** — گزارش تعداد دانلودها، ارسال‌ها و آرشیوها

---

## 🗂️ ساختار فایل‌ها

```
.
├── main.py            # نقطه شروع برنامه
├── config.py          # خواندن تنظیمات از .env
├── telegram_bot.py    # منطق اصلی ربات تلگرام
├── tg_client.py       # کلاینت Pyrogram برای دانلود فایل‌های بزرگ
├── tg_login.py        # لاگین یک‌بار به یوزر اکانت تلگرام
├── login_setup.py     # لاگین یک‌بار به حساب روبیکا
├── rubika_bot.py      # ارسال فایل به روبیکا
├── downloader.py      # دانلود از لینک (yt-dlp + مستقیم)
├── stats.py           # ثبت و نمایش آمار
└── passenger_wsgi.py  # فایل WSGI برای هاست‌های cPanel
```

---

## ⚙️ پیش‌نیازها

- Python 3.10+
- حساب تلگرام (برای یوزر اکانت)
- ربات تلگرام (از @BotFather)
- حساب روبیکا
- API از [my.telegram.org](https://my.telegram.org)

## 🚀 نصب و راه‌اندازی

### ۱. کلون کردن پروژه

```bash
git clone https://github.com/USERNAME/REPO_NAME.git
cd REPO_NAME
```

### ۲. نصب کتابخونه‌ها

```bash
pip install pyrogram tgcrypto python-telegram-bot rubpy python-dotenv jdatetime pyzipper yt-dlp
```

### ۳. ساخت فایل `.env`

یه فایل `.env` در ریشه پروژه بساز:

```env
TELEGRAM_TOKEN= توکن ربات 
TELEGRAM_API_ID= باید از my.telegram.org بگیرید
TELEGRAM_API_HASH= باید از my.telegram.org بگیرید
TELEGRAM_PHONE= 989185553322
ALLOWED_USER_ID= آیدی عدد تلگرام
FIXED_PASSWORD= رمز ثابت برای فایل های زیپ
```

> برای گرفتن `ALLOWED_USER_ID` پیام بده به [@userinfobot](https://t.me/userinfobot)

### ۴. تنظیم مسیر دانلود

در فایل `config.py` این خط رو پیدا کن و مسیر رو عوض کن:

```python
DOWNLOAD_DIR = "/home/YOUR_USERNAME/downloads/"
```

```bash
mkdir -p ~/downloads
```

### ۵. لاگین تلگرام (یک‌بار)

```bash
python tg_login.py
```

شماره‌ات رو وارد کن، کد SMS تلگرام رو بزن. فایل session ذخیره می‌شه.

### ۶. لاگین روبیکا (یک‌بار)

```bash
python login_setup.py
```

شماره روبیکات رو وارد کن، کد SMS رو بزن.

### ۷. اجرا

```bash
python main.py
```

**اجرا در پس‌زمینه:**

```bash
nohup python main.py > bot.log 2>&1 &
```

## 📦 وابستگی‌ها

| کتابخونه | کاربرد |
|---|---|
| `pyrogram` + `tgcrypto` | اتصال به یوزر اکانت تلگرام |
| `python-telegram-bot` | ربات تلگرام |
| `rubpy` | ارسال به روبیکا |
| `yt-dlp` | دانلود از یوتیوب، اینستاگرام و ... |
| `pyzipper` | فشرده‌سازی با رمز AES |
| `jdatetime` | تبدیل تاریخ به شمسی |
| `python-dotenv` | خواندن فایل `.env` |

---

## 🛠️ عیب‌یابی

| خطا | راه‌حل |
|---|---|
| `ModuleNotFoundError` | دوباره `pip install` بزن |
| ربات جواب نمیده | `ALLOWED_USER_ID` رو چک کن |
| خطای session | فایل `.session` رو پاک کن و دوباره لاگین کن |
| `PHONE_NUMBER_INVALID` | شماره رو با کد کشور وارد کن: `989...` |
| `DOWNLOAD_DIR` پیدا نشد | پوشه رو با `mkdir` بساز |
